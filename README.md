# 筋トレ記録

シンプルで速い、iPhone向けの筋トレ記録PWA（Webアプリ）です。

## 特長

- **記録**：種目・重量・回数・セットを素早く入力（±ステッパー／ドラムロール式の重量選択）
- **器具分類**：バーベル／ダンベル／マシン／ケーブル／自重で種目を管理・並べ替え
- **日次サマリー**：合計種目数・セット数・レップ数・負荷量を自動集計
- **種目情報**：鍛える部位の人体図、YouTubeでの動作検索、参考動画URL・フォームメモの登録
- **RM計算機**：ダイヤル式の重量入力から推定1RM換算表を表示
- **履歴**：月別に記録を閲覧
- **CSVエクスポート**：全記録をExcel対応のCSVで書き出し
- **オフライン対応**：ホーム画面に追加すればアプリのように使え、通信なしでも記録可能
- データは端末内（localStorage）に保存

## iPhoneでの使い方

1. iPhoneのSafariで公開URLを開く
2. 共有ボタン → 「ホーム画面に追加」
3. ホーム画面のアイコンから起動
